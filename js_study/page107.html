객체

객체 = 프로퍼티(값) + 메서드(행동)

내장 객체(교재 : Date, Math, Window, etc...) vs 사용자 정의 객체

프로퍼티 = 키 : 값

객체 = {중,괄,호}로 구성한다.

객체 중 키는 문자열, 숫자, 심볼 등

var student = {stu_name : "오동", gpa:4.5}

객체의 값에 접근: student.gpa ; student["stu_name"]

프로퍼티 수정 : student.gpa = 4.0;

프로퍼티 추가 : student.major = "없는 걸 쓰면 된다";

프로퍼티 삭제 : delete student.major

빈 객체 생성 : var person = { }; or var person = new Object();

복사 : 값 vs 참조

객체는 참조 복사(원본은 하나)

생성자 함수

객체 - 클래스 사용(ES6 규격)

Example - 원 넓이 구하는 객체

객체 이름 : CircleArea

프로퍼티 : 원 넓이 area, 반지름 radius

메소드 getArea : 원 넓이를 구해서 출력한다.

반지름 1to10 10개의 반지름을 갖는 원의 넓이를 구해보자
